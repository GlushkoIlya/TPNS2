# TPNS2
Лабораторные работы по теории и практике нейронных сетей.

## Содержание
Лабораторная №1 — препроцессинг и анализ датасета
Лабораторная №2.1 — MLP-регрессор
Лабораторная №2.2 — MLP-классификатор
Лабораторная №3 — рекуррентные нейронные сети (RNN, GRU, LSTM)

## Использованные датасеты
Laptop Price Dataset
Breast Cancer Wisconsin Dataset
Steel Industry Energy Consumption Dataset
